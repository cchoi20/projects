Notes:

  • Select a color from the color picker on the top-right.
  • Use the left mouse button to draw on the canvas.
  • Undo button allows you to undo the last drawn line.
      • You can also undo with [ctrl+z].
  • Clear button clears the entire canvas.
  • Save button saves the canvas as an image.
